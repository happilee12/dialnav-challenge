Overview

The DialNav Challenge is hosted at the Embodied Agent and Dialog (EAD) Workshop at ECCV 2026. The challenge benchmarks embodied agents that follow high-level natural-language instructions in indoor environments. Because the instructions are intentionally ambiguous, successful agents must proactively engage in dialog, ask clarifying questions, and disambiguate the intended path before — and while — acting.

💬

Proactive Dialog

Detect uncertainty and request clarification through natural conversation.

🧭

Navigation Under Ambiguity

Map vague high-level directions to concrete paths in unseen scenes.

🔄

Multi-turn Reasoning

Maintain dialog state and update beliefs as new information arrives.

📊

Dialog-Aware Metrics

Evaluation that balances task success, navigation efficiency, and question quality.

⌘

Task Description

What the agent must do

An agent receives a high-level instruction (e.g., "Go to the room where someone is reading") inside a photorealistic indoor scene. The instruction is deliberately under-specified. The agent must navigate to the correct goal location while being able to ask the human guide for clarification through dialog. Successful runs balance progress with concise, well-timed questions.

Evaluation

Success Rate (SR)

Fraction of episodes in which the agent stops within the goal threshold.

SPL

Success weighted by Path Length — efficiency-aware navigation success.

Question Efficiency

Number and informativeness of clarifying questions issued by the agent.

Dialog Quality

Human / model-based scoring of question relevance and grounding.
